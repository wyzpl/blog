# 项目简介

:::tip

运行此项目需要 **node 18** 以上

:::

**植树 DEV** 是个边学习边开发的个人练习项目。

此项目会学习积累一些 [**React**](/web/react/0)、**NextJS**、**MySQL** 相关经验，遇到的问题都会在对应的模块的笔记进行记录

主要技术栈：[**NEXT14**](https://nextjs.org/) + **[React18](https://react.dev/)** + **[TailWind CSS](https://tailwindui.com/)** + **MySQL** 

包管理工具：**[pnpm](https://www.pnpm.cn/)**
