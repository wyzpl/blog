## 路由组

在 `app`目录下，文件夹名称通常会被映射到 URL 中，但是可以将文件夹标记为路由组，阻止文件夹名称被映射到 URL 中。

把文件夹用括号括住就可以标记为路由组，比如 `(dashboard)`

使用路由组，你可以将路由和项目文件按照逻辑进行分组，但不会影响 URL 路径结构。路由组可用于比如：

1.  按站点、意图、团队等将路由分组
2.  在同一层级中创建多个布局，甚至是创建多个根布局
